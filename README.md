# vibeslurm

Vibe coded slurm monitor or something

![vibeslurm](media/vibeslurm.jpg)

### How to run

```bash
pixi run --manifest-path ./env/pixi.toml python -m vibeslurm
```

### Dependencies

* [QtPy](https://github.com/spyder-ide/qtpy) (with PyQt or PySide). Tested with PysSide6.
